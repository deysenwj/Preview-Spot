# Kampung Warna Warni Jodipan - Photo Spots Website

## Setup Instructions

### Folder Structure
Buat struktur folder seperti ini di komputer Anda:

```
kampung-jodipan/
├── kampung-jodipan.html
├── images/
│   ├── spot1.jpg
│   ├── spot2.jpg
│   ├── spot3.jpg
│   ├── spot4.jpg
│   ├── spot5.jpg
│   ├── spot6.jpg
│   ├── spot7.jpg
│   ├── spot8.jpg
│   ├── spot9.jpg
│   └── spot10.jpg
└── README.md
```

### Cara Menggunakan:

1. **Buat folder baru** bernama `kampung-jodipan` di komputer Anda
2. **Masukkan file HTML** (`kampung-jodipan.html`) ke dalam folder tersebut
3. **Buat folder `images`** di dalam folder `kampung-jodipan`
4. **Siapkan 10 foto** dari Kampung Warna Warni Jodipan
5. **Rename foto-foto** sesuai dengan nama berikut:
   - `spot1.jpg` - Main Gate of Jodipan Village
   - `spot2.jpg` - Rainbow Colorful Houses Area
   - `spot3.jpg` - Jodipan Colorful Bridge
   - `spot4.jpg` - 3D Koi Fish Mural Spot
   - `spot5.jpg` - Instagram Rainbow Stairs
   - `spot6.jpg` - Colorful Village Alley
   - `spot7.jpg` - Jodipan Flower Garden
   - `spot8.jpg` - Angel Wings Selfie Spot
   - `spot9.jpg` - Brantas River View
   - `spot10.jpg` - Colorful Umbrella Area

6. **Masukkan semua foto** ke dalam folder `images`
7. **Buka file HTML** dengan browser (double click `kampung-jodipan.html`)

### Format Foto yang Disarankan:
- Format: JPG, JPEG, atau PNG
- Aspek ratio: 16:9 (contoh: 1920x1080, 1280x720)
- Ukuran file: Maksimal 2MB per foto untuk loading cepat
- Resolusi minimal: 800x450 pixels

### Mengganti Foto:
Untuk mengganti foto, cukup replace file di folder `images` dengan nama file yang sama.
Misalnya untuk mengganti foto spot 1, ganti file `spot1.jpg` dengan foto baru Anda (tetap gunakan nama `spot1.jpg`).

### Font yang Digunakan:
- **Roboto Black** (weight: 900) - untuk judul dan nama spot
- **Roboto Regular** (weight: 400) - untuk subtitle
